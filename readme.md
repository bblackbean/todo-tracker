# Todo Tracker

Spring Boot 기반으로 만든 간단한 Todo API 프로젝트

## 기능
- 할 일 등록/조회/수정/삭제
- 검색, 필터링, 정렬, 페이징
- Validation, 예외 처리
- 로그인 및 회원가입
- 테스트 코드 (Controller, Service 계층)

## 사용 기술
- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- JUnit, AssertJ
- Spring Security
- Docker
